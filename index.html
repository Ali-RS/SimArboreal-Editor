    • Simarboreal-editor

    Editor using SimArboreal to generate exportable tree models.

    Project maintained by Simsilica Hosted on GitHub Pages — Theme by mattgraham

    Overview

    The SimArboreal Editor allows designers to create tree templates that can be used to export models directly or to create randomized procedurally generated trees based on the template.

    The SimArboreal library that this editor utilizes employs a simplified L-system that models tree and root branching.

    Another somewhat unique feature to this library is that it actually generates the roots of the tree in addition to the trunk and branches.
